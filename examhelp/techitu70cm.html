Authorized Technician Class Frequencies for 70 cm in ITU2

This frequency privilege question is a bit different. It specifies the ITU region you are operating in.  As it turns out, the United States is in ITU region 2, so you can just remember the value in the technician class band plan for 70 cm and you’ll know the correct answer is 443.350 MHz.

Another trick for this question:  443.350 MHz is the only choice that is in the 70 cm band.

Exam Question

[97.301(a)] Which 70 cm frequency is authorized to a Technician Class license holder operating in ITU Region 2?

A. 53.350 MHz

B. 146.520 MHz

C. 443.350 MHz

D. 222.520 MHz

More About It

Different ITU regions have different privileges in the 70cm band.  You can see the ITU frequencies on the ARRL web site’s ITU frequency table.
