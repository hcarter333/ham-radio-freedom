Parity Bit Error Detection

 

A parity bit is an additional bit sent with digital data to indicate whether or not an error occurred during the transmission of the data. The receiver of the digital data checks the parity bit when the data arrives to determine if there were any errors.

 

More About It

The value of a parity bit is determined by performing a simple calculation on the data about to be transmitted. The result of the calculation will change if the data changes. The receiver performs the same calculation on the received data and checks the result against the transmitted parity bit. For more about parity calculations, see the Wikipedia page on parity.
